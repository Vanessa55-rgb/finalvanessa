Acerca de Software

El software (o SW) es el conjunto de programas, instrucciones, reglas y datos que permiten que un sistema informático funcione. Es la parte no física, intangible, de un ordenador o dispositivo electrónico que se puede manipular y programar para realizar tareas específicas.
